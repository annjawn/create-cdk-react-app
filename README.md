## Overview

Just do

```sh
npx create-cdk-react-app my-new-app
cd my-new-app
```

Or specify language

```sh
npx create-cdk-react-app my-app --cdk typescript --react typescript
cd my-new-app
```

And you are off to the races into building your modern React web app with AWS backend.
